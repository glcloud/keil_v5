The Microcontroller Development Kit (MDK) supports software development for and debugging of Arm7, Arm9, Arm Cortex-M and Armv8-M processor-based devices. A detailed description of MDK can be found in the section Tools User's Guide of the µVision window Books.

What's New in the Microcontroller Development Kit

The next sections list the changes for each release of the MDK toolset.

MDK Version 5 is capable of using MDK Version 4 projects after installation of the Legacy Support. This adds support for Arm7, Arm9, and Arm Cortex-R4 processor-based devices.
